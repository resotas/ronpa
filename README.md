# 論破王ひろゆきチャットbot

このプロジェクトは、ユーザーの入力に対してひたすら論破を行う「ひろゆき風チャットbot」のLINE風Webアプリです。

---

## **機能**
- ユーザー入力に基づくひろゆき風の論破応答。
- LINE風のUIを再現。
- OpenAI APIを使用したダイナミックな応答生成。

---

## **プロジェクト構成**
- **React**: フロントエンドライブラリ
- **Vercel**: デプロイ環境
- **OpenAI API**: 応答生成
